# final-reports-checklist.md

## Gate estrito PTBE -> PE

Marcar todos os itens antes de migrar o AP para `plans-executed/`.

- [ ] `report-final.md` mestre completo.
- [ ] `report-final-execucao.md` completo.
- [ ] `report-final-arquitetura-e-contratos.md` completo.
- [ ] `report-final-riscos-e-pendencias.md` completo.
- [ ] Rastreabilidade para `architecting-plan/*` documentada.
- [ ] Rastreabilidade para subplanos executados documentada.
- [ ] Cobertura de `report-success.md` por subplanos executados.
- [ ] Cobertura de `report-merge.md` quando houve merge/conflito.
- [ ] Evidências de validação registradas.
- [ ] Pendências e riscos residuais registrados com contexto.
- [ ] Decisão formal de migração PTBE -> PE registrada no mestre.
