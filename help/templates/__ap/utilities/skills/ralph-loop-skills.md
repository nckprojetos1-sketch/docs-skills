# ralph-loop-skills

Lista utilitaria de skills relacionadas a `ralph-loop` no metodo DOCS.

## Skills encontradas (valor e quando usar)

1. `andrelandgraf/fullstackrecipes@ralph-loop`
   - Valor: referencia principal para fluxo `ralph-loop` com maior adocao.
   - Quando usar: quando voce quer um baseline mais consolidado para aplicar o loop.
2. `belumume/claude-skills@ralph-loop`
   - Valor: alternativa de implementacao do mesmo conceito.
   - Quando usar: para comparar abordagens e adaptar ao seu processo.
3. `2ykwang/agent-skills@ralph-loop-template`
   - Valor: template inicial para acelerar configuracao.
   - Quando usar: ao montar rapidamente uma variacao propria de `ralph-loop`.
4. `zenobi-us/dotfiles@ralph-loop`
   - Valor: versao integrada em colecao de automacoes pessoais.
   - Quando usar: quando voce prefere referencia pratica em setup de produtividade.
5. `anton-abyzov/specweave@ralph-loop-guide`
   - Valor: guia orientado a uso e entendimento do fluxo.
   - Quando usar: para onboarding ou consulta rapida de aplicacao.
6. `supercent-io/skills-template@ralph-loop`
   - Valor: variante baseada em template de skills.
   - Quando usar: para estudar estrutura e adaptar para um skill proprio.

## Instalacao individual

```bash
npx skills add andrelandgraf/fullstackrecipes@ralph-loop
npx skills add belumume/claude-skills@ralph-loop
npx skills add 2ykwang/agent-skills@ralph-loop-template
npx skills add zenobi-us/dotfiles@ralph-loop
npx skills add anton-abyzov/specweave@ralph-loop-guide
npx skills add supercent-io/skills-template@ralph-loop
```
