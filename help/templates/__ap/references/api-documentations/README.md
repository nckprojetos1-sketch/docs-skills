# api-documentations

## Objetivo
- Documentacao operacional de APIs no metodo DOCS, cobrindo exploracao, documentacao e validacao de contratos.

## Skills recomendadas
Use estas skills para entender APIs existentes, documentar APIs novas e validar comportamento/contrato.

1. `sickn33/antigravity-awesome-skills@api-documentation-generator`
- Valida porque: e especializada em gerar documentacao de API e possui alta adocao.
- Use quando: voce precisa criar documentacao de endpoints com exemplos rapidamente.
- Instalar: `npx skills add sickn33/antigravity-awesome-skills@api-documentation-generator`

2. `aj-geddes/useful-ai-prompts@api-reference-documentation`
- Valida porque: e focada em documentacao de referencia tecnica de APIs.
- Use quando: o objetivo e produzir documentacao tipo reference (params, respostas, erros).
- Instalar: `npx skills add aj-geddes/useful-ai-prompts@api-reference-documentation`

3. `hairyf/skills@openapi-specification-v2`
- Valida porque: foca em OpenAPI, padrao central para contrato e docs de APIs REST.
- Use quando: voce precisa criar ou melhorar o arquivo OpenAPI/Swagger da API.
- Instalar: `npx skills add hairyf/skills@openapi-specification-v2`

4. `personamanagmentlayer/pcl@openapi-expert`
- Valida porque: e especializada em qualidade de especificacoes OpenAPI.
- Use quando: voce quer revisar consistencia, cobertura e clareza do contrato OpenAPI.
- Instalar: `npx skills add personamanagmentlayer/pcl@openapi-expert`

5. `aj-geddes/useful-ai-prompts@api-contract-testing`
- Valida porque: trata diretamente validacao de contrato entre implementacao e especificacao.
- Use quando: voce precisa prevenir quebra de integracao entre producer e consumer.
- Instalar: `npx skills add aj-geddes/useful-ai-prompts@api-contract-testing`

6. `secondsky/claude-skills@api-testing`
- Valida porque: foca em testes funcionais de APIs.
- Use quando: voce quer validar status codes, payloads, erros e regressao de endpoints.
- Instalar: `npx skills add secondsky/claude-skills@api-testing`

7. `aj-geddes/useful-ai-prompts@rest-api-design`
- Valida porque: cobre boas praticas de design REST.
- Use quando: voce esta definindo recursos, verbos, versionamento e padroes de resposta.
- Instalar: `npx skills add aj-geddes/useful-ai-prompts@rest-api-design`

8. `apollographql/skills@graphql-operations`
- Valida porque: e especializada em operacoes GraphQL no ecossistema Apollo.
- Use quando: a API e GraphQL e voce precisa modelar/documentar queries e mutations.
- Instalar: `npx skills add apollographql/skills@graphql-operations`

## Instalacao em lote (opcional)

```bash
npx skills add sickn33/antigravity-awesome-skills@api-documentation-generator
npx skills add aj-geddes/useful-ai-prompts@api-reference-documentation
npx skills add hairyf/skills@openapi-specification-v2
npx skills add personamanagmentlayer/pcl@openapi-expert
npx skills add aj-geddes/useful-ai-prompts@api-contract-testing
npx skills add secondsky/claude-skills@api-testing
npx skills add aj-geddes/useful-ai-prompts@rest-api-design
npx skills add apollographql/skills@graphql-operations
```
