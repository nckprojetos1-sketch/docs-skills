# agent-audit

Gaveta de memoria operacional curta por plano AP.

Use esta pasta para registrar runs de planners, executor e reviewer ligados a
este AP ou aos seus subplanos.

## Formato

- `YYYY-MM-DD__agent-role__slug/`

Arquivos minimos por run:

- `run.md`
- `artifacts/` opcional

## Regra

- handoff nao fica aqui
- handoff fica em `reports/` do plano ou subplano
