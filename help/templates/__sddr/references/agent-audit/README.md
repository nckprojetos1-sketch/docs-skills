# agent-audit

Gaveta de memoria operacional curta por plano SDDR.

Use esta pasta para registrar runs de planner, executor ou reviewer ligados ao
plano.

## Formato

- `YYYY-MM-DD__agent-role__slug/`

Arquivos minimos por run:

- `run.md`
- `artifacts/` opcional

## Regra

- handoff nao fica aqui
- handoff fica em `reports/` do plano ou na resposta final do agente
